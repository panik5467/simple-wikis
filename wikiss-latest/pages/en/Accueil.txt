!!News

'''Beware, this is a beta version. It's inadvisable for production.'''

This 0.4beta1 version of ''WiKiss'' brings you some new functionalities :
* internationalization mechanism for WiKiss and a simple multilanguage plugin ([doc|http://wikiss.tuxfamily.org/?page=Langues]) ;
* a Menu plugin to display a common menu on every wiki pages ([doc|http://wikiss.tuxfamily.org/?page=svn:wkp_Menu]) ;
* a plugin to display configuration (plugins and locales) ([doc|http://wikiss.tuxfamily.org/?page=svn:wkp_ListPlugins]) ;
* an upload plugin. Allows you to easily upload files or pictures ([doc|http://wikiss.tuxfamily.org/?page=svn:wkp_Upload]) ;
* a lot of [bug|http://wikiss.tuxfamily.org/bts/] fixes.
See the complete [Changelog|http://wikiss.tuxfamily.org/?page=Changelog] for more details.
